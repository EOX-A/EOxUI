# Beercss font issue

- clone repo with `npm run clone:beercss`
- the script also copies the custom [`material-symbols-outlined.woff2`](./material-symbols-outlined.woff2) into the folder

This custom font file contains only the bare minimum characters to make beercss work (checkbox, radio, check mark). To generate it, visit https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@24,400,0..1,0&icon_names=check,check_box,check_box_outline_blank,indeterminate_check_box,radio_button_checked,radio_button_unchecked and downlaod the generated font link, save it as `material-symbols-outlined.woff2`.
